# NoImplicitAny
## É uma opção do compilador em typescript.
## É do tipo boolean (true ou false) por padrão é false.
## Da um erro em expressões e declarações com um tipo any implícito. Quando atribuimos valores à variáveis sem tipos definidos é comum que seja atribuido any como tipo da variável.
## Como declarar :
```
"compilerOptions":{
        "NoImplicitAny": true 
                  }
```
