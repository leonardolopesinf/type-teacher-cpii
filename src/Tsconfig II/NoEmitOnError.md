# NoEmitOnError.
## É uma opção do compilador em typescript.
## É do tipo boolean (true ou false) por padrão é false.  
## Não emite saídas se algum erro for relatado , isto é, se na hora de compilar tiver algum erro as saídas não irão ser enviadas.
## Como declarar :
```
"compilerOptions":{
        "NoEmitOnError": true 
                  }
```
